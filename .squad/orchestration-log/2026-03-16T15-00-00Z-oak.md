### Phase 3: Canvas Rendering Analysis
**Finding:** Main.ts line 12-15
```typescript
await app.init({
  background: GAME.BACKGROUND_COLOR,
  resizeTo: window,
});
```
Missing: `resolution: window.devicePixelRatio` and `autoDensity: true`  
Impact: All HiDPI displays (MacBooks, modern monitors, smartphones) render at 1x canvas into 2x-3x CSS pixel space = blurry.

### Phase 4: Event Bus & Memory Leak Check
**Finding:** GardenScene.init() lines 271-280, 417-424, 504
- Multiple `eventBus.on()` subscriptions
- No corresponding cleanup in `destroy()`
- After 3 visits: 3x event callback execution, 3x duplicate audio/visual effects
