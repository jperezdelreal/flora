- #200 "Smooth scene transitions"

## Work Summary

Implemented four distinct scene transition types with easing and input blocking:

### Transition Types Implemented
1. **Fade** — Linear opacity blend (0→1) over configurable duration
2. **Crossfade** — Dual-container simultaneous blend (old scene fades out while new fades in)
3. **Slide** — Position-based transition with translate animation and easing curves
4. **Loading** — Progress bar with parallel scene initialization

### Easing Functions
- `easeLinear` — constant speed
- `easeInOutCubic` — smooth acceleration/deceleration
- `easeOut` — quick start, slow finish
- `easeIn` — slow start, quick finish

### Input Blocking
- SceneManager sets `transitioning = true` during animation
- Skips `current.update()` calls while transitioning
- Input events silently dropped (not queued)

### Scene Routing
- Boot → Menu: `loading` ("Preparing the garden...")
- Menu → SeedSelection: `crossfade`
- SeedSelection → Garden: `fade`
- Menu → Garden (continue): `fade`
